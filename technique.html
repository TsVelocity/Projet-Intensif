La Technologie derrière Poll Urbain

Poll Urbain est construit en utilisant des technologies web modernes, notamment HTML, CSS et JavaScript. La plateforme utilise une interface utilisateur conviviale basée sur une simulation de carte de la ville, permettant aux utilisateurs de planifier et de visualiser la végétalisation de leur environnement urbain.

Quelle Technologie Utilisons-Nous ?

Nous utilisons une architecture web robuste avec un backend basé sur Node.js et une base de données MongoDB pour stocker les données de végétalisation. La communication entre le frontend et le backend se fait via des API REST, garantissant une expérience utilisateur fluide et réactive.
